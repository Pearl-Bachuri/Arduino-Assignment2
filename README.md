# Arduino-Assignment2
Arduino Uno project: beeping countdown using tone() and a seven segment display
This week's s my submission for Assignment 2 in the programming C++ for engineers using Arduino course at Ghana Communication Technology University  

## What this project does
A seven segment display counts down from 9 to 0. On each step, a passive buzzer piezo plays a short beep, but plays a longer one to signal completion of the code

## Hardware used
male to male jumper wires 
buzzer
7-segment display (common cathode)
Arduino Uno 
bread board

## Concepts demonstrated
the tone() and noTone() function
passive and active buzzers
7-segment display wiring (common cathode)
2D arrays for digit patterns
function with parameters
while and for loops
serial monitor output 

## How to run it
1. Wire yhe buzzer to pin 8.
2. Wire the seven segment a- g to Arduino pins 2,3,4,5,6,7,and 9
3. Connect the COM pin of the display to GND via a 220 ohm resistor
4. open the file in the Arduino IDE
5. select tools> board > Arduino Uno and the correct port.
6. click upload.
